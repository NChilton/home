命令提示符（cmd）右键菜单

发表于 2018-11-01 更新于 2021-12-08 分类于 配置技巧

很多时候我们需要在某个文件夹下打开cmd或者powershell，可以在Windows 运行（Winkey + R）里输入cmd或者powershell，打开窗口默认目录为C:\Users\<xxx>，但切换到目的目录比较麻烦。此处给出快速在当前目录打开cmd或者powershell的方法。

shift + 右键

在 Windows 下同时shift + 右键，则在右键菜单有在此处打开命令提示符窗口/Powershell 窗口，如下图所示（本人 Windows 10 为 1809 版）：

有几点需要注意：

  • 在 Windows 10 1703 版本之前，shift + 右键出现的是在此处打开命令提示符（cmd)，在此版本及之后出现的是在此处打开 Powershell 窗口。
  • 鼠标右击开始菜单，在 Windows 10 1703 版本之前，选项中会出现命令提示符和命令提示符（管理员），在此版本及之后出现的是Windows Powershell和Windows Powershell（管理员）。
  • 如果你的电脑安装了 Windows 子系统（WSL），则在 Windows 10 1809 版本及以后，shift + 右键会多出一个在此处打开 Linux shell选项。

修改注册表

shift + 右键中 2 区对应的注册表位置为计算机\HKEY_CLASSES_ROOT\Directory\Background\shell，将注册表位置定位到地址，其中数值名称为(默认)的数据值为右键菜单显示的名称，数值名称为ShowBaseOnVelocityld表示启用该选项（如下图 Powershell 就启用了），数值名称为HideBaseOnVelocityld表示不启用该选项（如下图 cmd 就没启用）。

安装 Bandzip 软件

Bandzip 是一款解压缩软件，此处极力安利。安装 Bandzip 软件后在选项中可以增加打开命令提示符...到右键菜单。

这种方式也是在注册表增加cmd选项，如下所示，1 表示启用，0 表示不启用。
